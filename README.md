# React + Vite
## How to Run
Ensure npm and node is installed: node -v, npm -v
npm install 
npm start 
## Project
Project is build to break scrolling reels addiction target is to reduce scrolling time. How many times have you deleted and reinstalled insta? Frontend of this looks exactly like instagram to help you gradually break the addiction keeping you connected with your friends and family. However in reels section as you scroll, gradually time of reel keeps increasing as you scroll without user actively realising, helping them gain back their attention span
